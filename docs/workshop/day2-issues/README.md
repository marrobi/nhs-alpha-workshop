# Day 2 Issue Templates

This directory is populated by the **Day 2 Issue Generator** agent at the start of Day 2.

The agent reads the actual tech stack, architecture, and codebase from Day 1 and generates tailored issue templates with precise acceptance criteria for each tool and framework in use.

## How to use

1. Switch to the **Day 2 Issue Generator** agent in VS Code
2. Ask it to generate the Day 2 issues
3. Review the generated files in this directory
4. The agent creates the issues on GitHub — you then assign Copilot with the correct custom agent for each issue

See [day2-guide.md](../day2-guide.md) for the full process.
