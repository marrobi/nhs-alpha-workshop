## Data Protection Impact Assessment

### Goal
Draft a DPIA for the service's processing of health data under UK GDPR.

### Acceptance Criteria
- [ ] `docs/dpia/` directory created
- [ ] DPIA document follows the ICO 8-step process
- [ ] Data flows documented (what personal data, where stored, who accesses)
- [ ] Lawful basis identified (likely GDPR Art. 6(1)(e) public task + Art. 9(2)(h) health care)
- [ ] Necessity and proportionality assessment completed
- [ ] At least 5 privacy risks identified and mitigated
- [ ] Data subject rights section completed (access, rectification, erasure, portability)
- [ ] DPO sign-off section included (even if placeholder for Alpha)
