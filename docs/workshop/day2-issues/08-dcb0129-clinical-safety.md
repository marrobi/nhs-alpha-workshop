## DCB0129 Clinical Safety Case

### Goal
Generate the clinical safety documentation required for NHS digital services handling clinical information.

### Acceptance Criteria
- [ ] `docs/clinical-safety/` directory created
- [ ] Hazard log created using the DCB0129 hazard log template
- [ ] At least 5 hazards identified using SIREN categories (incorrect data, delayed data, unavailable data, incorrect functionality, data integrity)
- [ ] Each hazard assessed with severity (1-5) and likelihood (1-5) using the NHS clinical risk matrix
- [ ] Mitigations documented for each hazard rated Moderate or above
- [ ] Clinical Safety Case Report drafted with scope, system description, hazard summary, and residual risk statement
- [ ] Report references the service's specific clinical context
